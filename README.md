# ProjetoAngulaar12
Uma interface Web que se comunica com uma API REST de dados de cadastros de empresas brasileiras. (Angular 12,MongoDB,Nodejs).

Desenvolvedores:
- Helena Lannes

### Começando o projeto:
- clone o repositorio
- cd angular-desafio
- ng serve --port 8081
- cd ../
- cd server_node
- npm start
